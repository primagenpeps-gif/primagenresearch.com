PRIMAGEN RESEARCH — IMAGE ASSETS
================================

Save your images in THIS folder with these EXACT names and they appear on the
site automatically — no code changes needed. Until a file exists, the site shows
a built-in fallback (SVG vial / text logo), so nothing ever looks broken.

REQUIRED
--------
1) assets/vials-hero.png   → the 3-vial photo (GLP-3 RT, BPC-157, GHK-Cu)
                             Shown as the main hero image on the first screen.

2) assets/vial.png         → the single-vial photo (GLP-3 RT)
                             Shown as the product image on every catalog card,
                             in the quick-view popup, and as the cart thumbnail.

3) assets/logo.png         → the full stacked Primagen Research logo (teal hex-P
                             + PRIMAGEN + RESEARCH). Shown on the footer and on
                             the Create Account / Sign In panel.

OPTIONAL
--------
4) assets/monogram.png     → just the hex-P mark, cropped square (e.g. 128x128).
                             If present, it replaces the simplified SVG mark in
                             the top navbar. If absent, the SVG mark is used.

5) assets/products/<id>.png → per-product photos (advanced). If you later want a
                             DIFFERENT image per product instead of the single
                             vial.png everywhere, send the photos and I'll wire
                             each product id (bpc157, tb500, glp3, ghkcu, …).

TIPS
----
• White or transparent backgrounds both work — product images sit on a clean
  light "studio" tile, so a white background blends in seamlessly.
• PNG (transparent) or high-res JPG are both fine. ~800px+ on the long edge.
• The hero and product tiles keep their aspect ratio (object-contain), so your
  images won't be stretched or cropped.
• File names are case-sensitive on some web hosts — keep them lowercase exactly
  as written above.
